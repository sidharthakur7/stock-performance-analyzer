📊 Stock Performance Analyzer

A Python-based tool to analyze stock performance using real-time market data.
This project calculates key financial metrics like return, volatility, and risk, and provides simple insights.

---

🚀 Features

- 📥 Fetch real-time stock data using "yfinance"
- 📊 Calculate:
  - Total Return
  - Average Daily Return
  - Volatility (Risk)
  - Maximum Gain & Loss
- 🧠 Generate simple insights (risk level, performance)
- 🔍 User input for dynamic stock analysis

---

🛠️ Technologies Used

- Python
- pandas
- numpy
- yfinance

---

📦 Installation

1. Clone the repository:

git clone https://github.com/sidharthakur7/stock-performance-analyzer.git
cd stock-performance-analyzer

2. Install required libraries:

pip install yfinance pandas numpy matplotlib

---

▶️ Usage

Run the script:

python main.py

Enter a stock symbol when prompted:

AAPL
RELIANCE.NS
TCS.NS

- Average Daily Return: 0.05%
